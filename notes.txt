gitHub

1) Консоль 
Git просит сначала представиться и предлагает указать email и имя, 
чтобы в коммитах это отображалось и по умолчанию использовались эти данные.

Нужно ввести что предлагает гит:

git config --global user.email "you@example.com"
git config --global user.name "Your Name"

dir что есть в папке
cd смена директории
git init - инициализировать папкус проэктом

cd.. выйти на уровень выше

git status - статус

git add название файла - git будет отслеживать этот файл

git add . = добавить все

git rm --cashed название файла - перстать отсеживать файл

git commit -m Название комита рандомное

.gitignore - папка для игнора котрую нужно добавить git add .

git branch - ветка гита

git branch Название ветки - создание ветки

git branch -D название ветки - удаление ветки

git push - залить с локалки на репозиторий

git checkout название ветки - переключение на другую ветку

git checkout -b название ветки - создание ветки и сразу работа в ней

git merge название ветки - которую нужно совеместить с мастером
